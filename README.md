## AWESOME DINO
Game for CS
### Features
   - Obstacles manager
   - Infinitely moving ground
   - Jump
   - Score & High Score _(W/R file)_
   - Faster fall
   - Leveling system
   - Pause
   - **EASTER EGG** _(search for it in intro screen)_

### Keybinds:
##### Jump: `SPACE`, `w`, `ARROW UP`
##### Fall: `s`, `ARROW DOWN`
##### Pause: `p`, `ESC`
##### Debug mode: <code>` (backtick)</code>
